How to Setup on Live Server

Step 1 - Upload Files
  • Login to your CPanel, you can access that from your Hosting Service Provider.
  • Click on "File Manager".
  • From the bunch of folder, find public_html, folder
  • Upload the zip archive upload_this.zip inside public_html.
  • Extract the Contents of zip inside public_html.
Step 2 - Create A Database
  • From your CPanel, click on MySQL Database
  • Create a New Database and a New User.
  • Take Note of credentials will be required later.

How to Setup on Localhost

For Local Development your must be running XAMPP or WAMP on your machine.

Step 1 - Copy Files
  • Create a folder in your local server, name it "myfolio". Inside htdocs folder if you are using XAMPP or www folder if you are using WAMP.
  • Copy and paste here the archve file upload_this.zip, that you have downloaded.
  • "Right Click" on it.
  • Extract the contents inside this folder.
Step 2 - Create a Database
  • Start your Apache Server and MySQL Server
  • navigate to phpMyAdmin area, to manage database.
  • Click on New, and enter a name for your database then create it.
  • Now create a User for your database, under User Accounts tab.
  • Create a New User; keep a track of username and password.
Step 3 - Create a Local Domain (Advanced & Optional)
  • Open Notepad with Administrative Privileges
  • Click on File and Open.
  • Navigate to C:\Windows\system32\drivers\etc, select "All Files", then open Hosts file;
  • Now simply add, 127.0.0.1    myfolio.local, to the bottom and save it.
Step 3.1 - Setup VirtualHost on Apache Server
  • Navigate to C:\xampp\apache\conf\extra, find and open httpd-vhosts.conf in notepad.
  • Without touching anything, add this piece of code at bottom:
    •                                 <VirtualHost *:80>
      ServerAdmin webmaster@dummy-email.com
      DocumentRoot "C:/xampp/htdocs/myfolio"
      ServerName myfolio.local
      ErrorLog "logs/myfolio-error.log"
      CustomLog "logs/myfolio-access.log" common
    </VirtualHost>
  • Save and Exit.

Note: After finishing this steps now you can visit and access MyFolio at myfolio.local. As this Step 3 is optional, its upto you if you want to setup a domain or not, without this you can access MyFolio at localhost/myfolio

How To & FAQ

Some Important Notes and frequently asked questions.

How to add Cron Job and why it is required?

A cron job is a task scheduler in Unix-like systems that run shell commands at certain intervals. It is a key concept in Laravel and is used to run administrative tasks periodically on the server, such as sending emails or cleaning cache in file system.

In Myfolio, Task Scheduler is used to run Queue Worker, periodically. As your site grows you might receive a high traffic and newsletter subscribers. Now, when you send newsletter mail at once server response time can't handle it. So, Myfolio add all emails to queue and will be sent in background. There is where CRON Job will come to action, and start the worker, and worker checks if there is any task in queue, if there is no task it will stop automatically.

  • Login into your CPanel of Your Hosting Provider.
  • Find CRON Job, mostly in Advanced Tab.
  • Depending on your Hosting Provider, the ui and options may be different for you.
  • In Common Options set Once Per Minute (* * * * *).
  • In Command to Run set [route to php] [route to cron script]
  • /usr/bin/php /home/rcubedev/public_html/artisan schedule:run
  • This will run the schedule:run command every once per minute.
How to Edit Code?
MyFolio maintains Laravel Structure (MVC).
  • All Routes or URL Links used in this project are defined at root_of_project/routes/web.php
  • All Models Controllers, Livewire Componens are located at root_of_project/app/http/
  • All Views and UI are located at root_of_project/resources/views
  • Frontend Themes are located at root_of_project/themes/

Note: Editing components or code may brick you application. Without knowledge I don't recommened you to do it.
